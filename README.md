# JD Engine
## A custom game engine written in C++ using Vulkan.
This is a work in progress engine that will be built up over time.
My aim to to make a simple engine that is abstracted enough that games can be built on top of it without needing to worry about the underlying implementation.


Libaries used:
- Vulkan
- GLFW
- GLM
- VKBootstrap
- TinyGLTF
- Jolt
